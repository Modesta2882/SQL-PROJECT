Project Title: Mastering SQL: HackerRank Solutions
Project Overview:
A documentation of my journey through the HackerRank SQL track, ranging from basic data retrieval to advanced database manipulation. This project serves as a technical portfolio of my ability to query relational databases effectively.

Repository Structure:

/Basic: Simple SELECT statements and basic filtering.

/Intermediate: Aggregations, Joins, and working with NULL values.

/Advanced: Complex subqueries, window functions, and mathematical functions.
